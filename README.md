# Bill Generator System

## Overview

The **Bill Generator System** is a Python-based application that helps generate bills for purchased items. It allows users to enter product details such as item name, quantity, and price, and automatically calculates the total cost.

This project demonstrates basic programming concepts such as user input handling, arithmetic operations, and structured program design using Python.

---

## Features

* Add multiple items to a bill
* Calculate total cost automatically
* Display itemized bill summary
* Simple and easy-to-use interface
* Handles multiple product entries

## How It Works

1. User enters product details.
2. Program calculates total cost based on quantity and price.
3. The system generates a final bill summary.

Example:

```
Item: Pen
Quantity: 2
Price: 10

Item: Notebook
Quantity: 1
Price: 50

Total Bill: 70
```
